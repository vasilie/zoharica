Sublime Text 3 - Windows Shortcuts

Save Ctrl+S
Join lines Ctrl+J
Goto Anything Ctrl+P
Goto line in a file Ctrl+P+:(line number)
Goto Definition in a file Ctrl+P+@(definition)
Distraction Free Mode Shift+F11
Comment out Ctrl+/
Select entire line Ctrl+L
Select similary indented code Ctrl+Shift+J
Select Everything inside HTML tag Ctrl+Shift+A
Paste with identation Ctrl+Shift+V
Switching lines Ctrl+Shift+UP/DOWN
Delete whole line Ctrl+Shift+K
Advanced New File * Ctrl+Alt+N
Command Palette Ctrl+Shift+P
Indent Line Ctrl+Shift+[ or ]
Console Ctrl+`

*Advanced New file requires Advanced New file Plugin

Find All Alt+F3
Find Next Ctrl+D
Incremental Find Ctrl+I
